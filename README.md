# BankCustomerChurn
## This README is still under revision, but please refer to the summary below for now

Dataset: 10,000 labeled bank customer data

Schema:

1. RowNumber	
2. CustomerId	
3. Surname	
4. CreditScore	
5. Geography	
6. Gender	
7. Age	
8. Tenure	
9. Balance	
10. NumOfProducts	
11. HasCrCard	
12. IsActiveMember	
13. EstimatedSalary	
14. Exited 

Goal: classify churn or not churn (Exited)

Pipeline:
1. Read data
2. Data preprocessing
3. Model training and testing (random forest, KNN, and logistic
regression)
4. Model evaluation
